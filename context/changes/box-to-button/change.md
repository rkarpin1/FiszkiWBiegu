## Cel

Zastąpić wszystkie miejsca, gdzie `Box`, `Row` lub `Column` z modyfikatorem `.clickable` pełnią rolę przycisku, właściwymi komponentami Compose (`Button`, `IconButton`, `FloatingActionButton` itp.) — bez zmiany wyglądu aplikacji.
